Org_Heigl_Ghostscript

Description
Description | Vars (details) | Methods (details) Constants (details)

This class contains a wrapper around the Ghostscript-Application.

This needs the Ghostscript application to be installed on the server. If the gs-executable is not available the class will not be able to execute anything

A working example might look like the following code:

  1.  // First we describe the output-format
  2.  $device = new Org_Heigl_Ghostscript_Device_Jpeg ();
  3.  
  4.  // Set the JPEG-Quality to 100
  5.  $device -> setQuality ( 100 );
  6.  
  7.  // Next we Create the ghostscript-Wrapper
  8.  $gs = new Org_Heigl_Ghostscript ();
  9.  
  10.  // Set the device
  11.  $gs -> setDevice ( $device )
  12.  // Set the input file
  13.      -> setInputFile ( 'path/to/my/ps/or/pdf/file' )
  14.  // Set the output file that will be created in the same directory as the input
  15.      -> setOutputFile ( 'output' )
  16.  // Set the resolution to 96 pixel per inch
  17.      -> setResolution ( 96 )
  18.  // Set Text-antialiasing to the highest level
  19.      -> setTextAntiAliasing ( Org_Heigl_Ghostscript::ANTIALIASING_HIGH );
  20.  
  21.  // convert the input file to an image
  22.  if ( true === $gs -> render () ) {
  23.      echo 'success';
  24.  } else {
  25.      echo 'some error occured';
  26.  }

Alternatively the example could read as follows

  1.  // Create the ghostscript-Wrapper
  2.  $gs = new Org_Heigl_Ghostscript ();
  3.  
  4.  // Set the device
  5.  $gs -> setDevice ( 'jpeg' )
  6.  // Set the input file
  7.      -> setInputFile ( 'path/to/my/ps/or/pdf/file' )
  8.  // Set the output file that will be created in the same directory as the input
  9.      -> setOutputFile ( 'output' )
  10.  // Set the resolution to 96 pixel per inch
  11.      -> setResolution ( 96 )
  12.  // Set Text-antialiasing to the highest level
  13.      -> setTextAntiAliasing ( Org_Heigl_Ghostscript::ANTIALIASING_HIGH );
  14.  
  15.  // Set the jpeg-quality to 100
  16.  $gs -> getDevice () -> setQuality ( 100 );
  17.  
  18.  // convert the input file to an image
  19.  if ( true === $gs -> render () ) {
  20.      echo 'success';
  21.  } else {
  22.      echo 'some error occured';
  23.  }
